## 📌 概要
本リポジトリでは、主に以下のコンテスト形式の回答を管理しています。

### 🔵 AtCoder Beginner Contest (ABC)
- **構成:** A〜G問題（計7問） / 100分
- **特徴:** 計算量の見積もりと素早い実装力が求められるコンテスト。
- **スタンス:** 計算幾何や典型アルゴリズム（動的計画法、グラフ理論等）をC#で高速に実装することを目標としています。

### 🔴 AtCoder Heuristic Contest (AHC)
- **構成:** A問題（1問） / 数時間〜数日間
- **特徴:** 最適解のない問いに対し、山登り法や焼きなまし法を用いて「より良い解」を追求するコンテスト。
- **スタンス:** 材料工学専攻で培った論理的思考を活かし、パラメータ調整や効率的な遷移設計に注力しています。

### 🟡 AtCoder Regular Contest (ARC)
- **構成:** 数問構成 / 120分
- **特徴:** 高度な数学的考察と、一捻りある独創的な発想が要求される上級者向けコンテスト。

## 🛠 開発・提出環境
効率的なコーディングと管理のため、ツールによる自動化を行っています。

- **Language:** C# (.NET 8.0)
- **Library:** [atcoder-library-csharp](https://github.com/Csharp-Solutions/atcoder-library-csharp) (ACLのC#移植版)
